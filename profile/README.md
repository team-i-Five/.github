## 🎭 mumo Project

### 개인 맞춤 뮤지컬 추천 서비스 mumo
사이트 링크 : 3개의 서버를 사용하는 프로젝트이기에 현재 웹 배포는 중단되었습니다.\
로컬 컴퓨터에서 테스트 할 수 있는 환경 조성을 위한, `Web, Ml, Data` 각각의 `DockerFile`가 폴더 별로 첨부되어있고,\
`MySQL DB`를 생성하여 첨부해둔 데이터셋을 이용해서 테스트 해볼 수 있습니다.
